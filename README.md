### 一些心得

- 行为金融学指出，当某种资产被广泛地误价而偏离产生“套利”时，修正这种套利的策略可能非常有风险，称这一现象为套利限制(limits of arbitrage)。
- 相信频繁的买卖是没有好处的，把资产价值当作鞅（或者下鞅）来看待。**谨记**
